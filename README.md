# python-web-unittest-profile-line-graph

## Description
A demo of creating a line graph.
Tested with unittest and selenium
chrome driver.

Docker compose demo of profile flag.

## Tech stack
- numpy
- matplotlib
- pandas
- unittest
- selenium

## Docker stack
- python:latest
- selenium/standalone-chrome

## To run
`sudo ./install.sh -u`
[Availble here](http://localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
